# 🚀 启动说明

## 安装依赖

在项目目录 `bgm_web` 下执行：

```powershell
npm install
```

## 启动开发服务器

```powershell
npm run dev
```

服务器将在 `http://localhost:3000` 启动并自动打开浏览器。

## 使用流程

### 1️⃣ 添加游戏
- 点击右侧 **"添加游戏"** 按钮
- 输入游戏名称搜索（如：原神、塞尔达传说、艾尔登法环）
- 点击 **"添加"** 按钮将游戏加入游戏库

### 2️⃣ 放置游戏
- 在右侧游戏列表中点击游戏卡片
- 游戏会出现在左侧坐标系中央
- 拖动游戏到合适的位置

### 3️⃣ 调整大小
- 鼠标悬停在游戏上
- 拖动右下角的缩放图标

### 4️⃣ 管理游戏
- 右键点击游戏可以移除或重置大小

## 坐标轴说明

- **横轴 X**：不好玩（左） ← → 好玩（右）
- **纵轴 Y**：不爱玩（下） ← → 爱玩（上）

## 常见问题

**Q: 为什么搜索不到游戏？**
A: 确保输入的是游戏的正式名称，可以尝试中文名或日文名。

**Q: 游戏无法拖动？**
A: 请确保在游戏封面区域按住鼠标左键拖动。

**Q: 如何删除已添加的游戏？**
A: 在坐标系中右键点击游戏，选择"移除"即可。
