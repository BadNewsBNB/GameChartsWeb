# 🚀 快速部署指南

将游戏排名图快速部署到线上，最快5分钟完成！

## ⚡ 最快方式：Vercel（推荐）

### 步骤1：准备GitHub仓库

```bash
# 初始化Git（如果还没有）
git init

# 添加所有文件
git add .

# 提交
git commit -m "准备部署"

# 推送到GitHub（需要先在GitHub创建仓库）
git remote add origin https://github.com/你的用户名/你的仓库名.git
git push -u origin main
```

### 步骤2：部署到Vercel

1. 访问 https://vercel.com
2. 点击 "Sign Up" 用GitHub账号登录
3. 点击 "New Project"
4. 选择你刚才创建的GitHub仓库
5. 配置设置：
   - Root Directory: `bgm_web`
   - Build Command: `npm run build`
   - Output Directory: `dist`
6. 点击 "Deploy"

### 步骤3：等待部署完成

大约1-2分钟后，你会得到一个网址，例如：
```
https://你的项目名.vercel.app
```

**就这么简单！**✅

## 🔄 后续更新

以后每次你推送代码到GitHub，Vercel会自动重新部署：

```bash
git add .
git commit -m "更新内容"
git push
```

等待1-2分钟，新版本就上线了！

## 🌐 使用自定义域名（可选）

1. 在Vercel项目设置中点击 "Domains"
2. 输入你的域名（如 `game.yourdomain.com`）
3. 按照提示在你的域名DNS设置中添加CNAME记录
4. 等待DNS生效（通常几分钟到几小时）
