# 🎮 游戏排名可视化工具

一个基于 Vue 3 的游戏评价可视化应用，通过四象限坐标系直观展示你对游戏的评价。

**在线体验**: [games.marblephantasm.org](https://games.marblephantasm.org)

## ✨ 功能特性

- 🎯 **四象限坐标系** - 在自由定义的四个维度评价游戏
- 🔍 **游戏搜索** - 通过 Bangumi API 搜索海量游戏数据
- 📦 **批量导入** - 从 Bangumi 导入你的游戏收藏
- 🖼️ **拖拽上传** - 直接拖拽本地图片添加自定义游戏
- 🖱️ **自由编辑** - 拖拽调整位置，缩放调整大小
- 🎨 **自定义设置** - 修改坐标轴标签、图表标题、颜色等
- 📸 **导出图片** - 一键导出高清排名图
- 💾 **本地存储** - 数据自动保存到浏览器，无需注册

## 🚀 快速开始

### 本地运行

```bash
# 克隆项目
git clone https://github.com/BadNewsBNB/GameChartsWeb.git
cd GameChartsWeb

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 `http://localhost:3000` 即可使用。

### 构建部署

```bash
# 构建生产版本
npm run build

# 输出目录：dist/
```

## 📖 使用指南

### 1️⃣ 添加游戏

#### 方式一：搜索添加
1. 点击右侧 **"添加条目"** 按钮
2. 输入游戏名称（支持中文/英文/日文）
3. 从搜索结果中选择并添加

#### 方式二：批量导入
1. 点击右侧 **"导入收藏"** 按钮
2. 输入 Bangumi 用户名
3. 选择想要导入的游戏

#### 方式三：拖拽图片
1. 直接从文件管理器拖拽游戏封面图片到右侧列表区域
2. 图片会自动添加为游戏项（名称为空，可后续编辑）

### 2️⃣ 放置到坐标系

- **点击游戏卡片** - 游戏会出现在坐标系中心
- **拖动游戏** - 按住游戏封面拖动到合适位置

### 3️⃣ 调整和编辑

#### 调整大小
- **单个调整**：鼠标悬停在游戏上，拖动右下角的缩放手柄
- **批量调整**：使用左下角的"图标大小"控制按钮

#### 整体缩放
- 使用左下角的"整体缩放"控制按钮调整画布显示比例

#### 右键菜单
- **置顶** - 将游戏移到最上层
- **重置大小** - 恢复默认尺寸
- **移除** - 从坐标系移除（返回待选库）

### 4️⃣ 自定义设置

点击左上角 **"图表设置"** 按钮：

- **坐标轴标签** - 修改横轴/纵轴的正负标签文字
- **图表标题** - 添加标题、调整字体大小、颜色、位置
- **导入/导出数据** - 备份或迁移你的排名数据

### 5️⃣ 导出图片

1. 点击右上角 **"导出图片"** 按钮
2. 自动生成高清 PNG 图片

## 💡 使用技巧

1. **快速置顶**：点击已在坐标系中的游戏可快速置顶
2. **批量操作**：使用左下角的控制面板批量调整所有游戏大小
3. **数据备份**：定期在设置中导出数据，防止浏览器清理缓存导致丢失
4. **自定义图片**：拖拽上传的图片支持所有操作，适合添加 Bangumi 没有的游戏

## 📝 常见问题

**Q: 数据会丢失吗？**  
A: 数据保存在浏览器 localStorage 中，除非清理浏览器数据，否则会一直保留。建议定期导出备份。

**Q: 可以添加 Bangumi 没有的游戏吗？**  
A: 可以！直接拖拽游戏封面图片到右侧列表即可添加自定义游戏。
